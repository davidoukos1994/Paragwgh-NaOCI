# NaOCl Storage v7

Έκδοση χωρίς «χωρητικότητα στόχου».

Η κάρτα σύνοψης **Σύνολο ποσότητας δεξαμενών** δείχνει μόνο το άθροισμα των πραγματικών ποσοτήτων που δηλώνεις σε κάθε δεξαμενή:

`Πραγματικά m × tn/m` για κάθε δεξαμενή και μετά άθροισμα όλων.

Ανέβασε όλα τα αρχεία στο GitHub Pages. Αν βλέπεις παλιά τιμή, κάνε refresh ή διέγραψε προσωρινά τα δεδομένα από το κουμπί Reset.
