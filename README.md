# Hypochlorite Storage Calculator

Απλή εφαρμογή για GitHub Pages. Υπολογίζει σε πόση ώρα γεμίζει η επιλεγμένη δεξαμενή NaOCl με βάση την παραγωγή kg/hr.

## Δεδομένα δεξαμενών

- Z2: max 7.50 m, 13.54 tn/m
- Z1: max 8.65 m, 11.63 tn/m
- Z3: max 7.50 m, 13.54 tn/m
- D2: max 6.20 m, 11.63 tn/m
- D3: max 6.20 m, 11.63 tn/m
- D1: max 6.20 m, 11.63 tn/m

Το σταθερό στοιχείο είναι το `tn/m`. Το `Max m`, το `Πραγματικά m τώρα`, η παραγωγή `kg/hr` και η δεξαμενή που γεμίζει αλλάζουν από την οθόνη.

## Τρέξιμο τοπικά

Άνοιξε το `index.html` με browser.

## Ανέβασμα στο GitHub Pages

1. Δημιούργησε νέο repository στο GitHub.
2. Ανέβασε τα αρχεία `index.html`, `style.css`, `app.js`, `README.md`.
3. Πήγαινε Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Save.

Μετά από λίγα λεπτά θα έχεις link για την εφαρμογή.
